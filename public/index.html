To replicate memory leak:

  1. Open memory tab in devtools and take a snapshot
  2. Press the "Leak!" button below which creates 1000 vue instances
  3. Take another snapshot and set the type from "Summary" to "Comparison"
  4. Note that there are still 1000 instances of both VueI18n and Vue in memory
  5. This does not happen when VueI18n is removed
